UNITTEST HOWTO

* generate a test_MODULE.py file for each MODULE
* in each file generate a TestFUNCTION class, which is intended to test the FUNCTION in the MODULE
* generate test_* functions in oder to test different test cases for the individual FUNCTION


To RUN all available tests:
* open a console in the scikit-comm folder 
* run: python -m unittest
* OR
* open a console in the scikit-comm folder and specify the start directory (-s) and the top-level-directory (t-) in verbose option (-v)
* python -m unittest discover -s skcomm/tests -t . -v
